* Emacs

** Building from source

I recommend visiting [[https://aur.archlinux.org/packages/emacs-git][emacs-git]] on aur, to get an idea of which packages you (or me in the future) will need to build emacs from source.

Then:

#+begin_src bash
  git clone https://git.savannah.gnu.org/git/emacs.git
  ./autogen.sh
  ./configure --with-native-compilation=aot --with-pgtk --with-tree-sitter --with-imagemagick --with-mailutils
  make -j$(nproc)
  make install
#+end_src

** Setting Up

After emacs is installed you can create the emacs config directory:

#+begin_src bash
  mkdir -p ~/.config/emacs
#+end_src

Then open the [[file:emacs.org][emacs.org]] in emacs and tangle it (~C-c C-v C-t~)!
